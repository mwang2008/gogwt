What is GoGWT Tracker
GoGWT Tracker is an application for Android powered phones which your current location and messaging activities(send/receive) can be viewed remotely through the internet by your parents and/or your guardians and/or your friends. You can view history tracker easily with web access.

Basic functions/features of GoGWT Tracker
GoGWT tracker contains two parts: client the Android phone, and server web application.

  • Enrollment
    • In order to start tracking/viewing tracks, you need to create an account. Enrollment can be obtained by either from web site enroll or from Android phone .
  • Login
    • After enrollment, with your phone you can login with GroupId and Display Name (any name that identify the track you are going to start with) and start tracking.
  • Settings
    • The setting can specify preferred units (Miles vs. Kilometers), whether send message body when sending message contents to the server etc.

For server side features, please refer gogwt server for details.
